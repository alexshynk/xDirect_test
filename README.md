Тестовое задание:  
Необходимо реализовать LP согласно вложенному дизайну(xDirect.jpg).  
Задание необходимо сделать в два этапа:  
1. Сделать верстку страницы с адаптацией под планшеты и мобильные устройства  
2. Сделать программное выполнение, а именно:   
- проверка корректности ввода данных в форме;   
- данные из формы должны писаться в базу данных;   
- пользователю должно отправится письмо с информацией;   
- верстка и программное выполнение на LP, должны быть разделены (так чтоб можно было спокойно вносить изменения в программную часть или сменить дизайн полностью или частично)  
  
Примечание: При верстке желательно использовать LESS   
  
-------------------------  
Реализация  
 - Дизайн сделан подобным, но картинки другие  
 - Адаптивный дизайн реализован средствами css  
	!Использование библиотеки LESS сведено к отображению иерархии в стилях элементов (путем вложености) как на странице  
 - Для просмотра добавленных пользователей - кликнуть на логотипе  
   
Описание файлов:  
index.html - главная страница  
db.php - конмтанты доступа к БД  
/css/style.css - папка со стилями  
/img - папка с подгружаемыми картинками  
/js/contact_form.js - javascript для главной страницы  
/php/common.php - в скрипте функции доступа к БД   
/php/send.php - скрипт регистрации пользователя  
/php/show_contacts.php - скрипт для отображения зарегистрированных пользователей  
  
-------------------------  
Описание установки  
1. Содержимое папки www скопировать в каталог доступный для WEB сервера  
2. Создать[выделить] БД для приложения  
3. Загузить скрипт main.sql  
4.Прописать в файл db.php актуальные параметры доступа БД
